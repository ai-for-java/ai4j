# ai4j

Java library for smooth integration with popular AI tools and services

## How to use
Maven:
```
<dependency>
  <groupId>dev.ai4j</groupId>
  <artifactId>ai4j</artifactId>
  <version>0.0.1</version>
</dependency>
```

Gradle:
```
implementation 'dev.ai4j:ai4j:0.0.1'

```
